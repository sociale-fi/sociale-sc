# Legacy

This directory preserves the smart contract implementation submitted to the **Cardano Blockchain Championship 2025**.

These files are kept as a reference point for the implementation history and are not part of the current production codebase.

## Contents

- `contracts.md/` — Original contract documentation from the competition submission. Describes a more complete API than was implemented at the time; useful as a reference for planned functionality.
- `examples/escrow.ak` — Escrow validator developed alongside the competition contracts. Not integrated with the core cooperative contracts; retained as a useful Aiken reference.

Current contracts live in `validators/` and `lib/` at the repository root.
